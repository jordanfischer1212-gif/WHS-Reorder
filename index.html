Reorder Tracker

Log an order when it goes out. This tracks quantity and cadence and tells you when the next one is due. Synced live with Firebase — anyone on this page sees the same data.

Need help? Click here for instructions

Adding a new order

Pick the hotel account (or choose "+ Add new account" if it's not listed yet), the product, quantity, and how often it should be reordered. If you're adding a new account, include a reminder email — that's who gets notified 3 days before each order is due.

Marking an order sent

Once you've actually placed/shipped an order, click Mark sent today on that row. This logs it and automatically moves the next due date forward based on that product's reorder cadence.

Pause and Cancel

  • Pause stops reminders for that order without losing its settings — use this for a temporary hold. A Resume button brings it back.
  • Cancel removes it from the active list entirely. You'll be asked to confirm first, since it's harder to undo.

Email reminders

Each account's contact automatically gets an email 3 days before their next order is due. That email includes Pause this order and Cancel this order links — if a customer clicks one, it opens a pre-filled email to customer service so we can make the change here manually. Nothing pauses or cancels itself automatically.
